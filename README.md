# Tech Summit Planner Web App

## Overview

The **Tech Summit Planner Web App** is a simple frontend web application that helps users organize sessions for a technology summit.
Users can enter a session title and speaker name to plan the event schedule.

This project demonstrates basic **web development concepts** using HTML, CSS, and JavaScript.

## Features

* Add new summit sessions
* Enter speaker details
* Simple and interactive interface
* Neon-themed UI design
* Easy to run in any web browser

## Technologies Used

* **HTML** – Structure of the web page
* **CSS** – Styling and neon design
* **JavaScript** – Basic functionality and interaction

## Project Structure

```
tech-summit-planner
 ├── index.html
 ├── style.css
 └── script.js
```

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.

## Example Usage

1. Enter a **Talk Title** (e.g., "AI in Business Analytics").
2. Enter a **Speaker Name** (e.g., "John Smith").
3. Click the **Inject** button to add the session.
